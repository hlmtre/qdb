Getting all quotes:
qdb.zero9f9.com/api.php?q=quotes&n=all
returns JSON array of quote objects
[{"0":"263","id":"263","1":"\"i am some quote\"","quote":"\"i am some quote\"","2":"2014-03-14 12:49:26","date":"2014-03-14 12:49:26"},{"0":"262","id":"262","1":"\"i am some quote\"","quote":"\"i am some quote\"","2":"2014-03-14 12:49:04","date":"2014-03-14 12:49:04"}]

Getting one quote:
qdb.zero9f9.com/api.php?q=quote&id=id_here
returns JSON quote object
{"0":"263","id":"263","1":"\"i am some quote\"","quote":"\"i am some quote\"","2":"2014-03-14 12:49:26","date":"2014-03-14 12:49:26"}

Submitting a quote:
POST to qdb.zero9f9.com/api.php, POST form field 'q' set to 'new', POST form field 'quote' set to the quote you want to submit.
returns JSON status object
a JSON object including the ID of your newly-submitted quote and some status information.

Deleting a quote (requires a user be an admin):
GET to qdb.zero9f9.com/api.php?q=delete&id=id_here. GET field 'q' set to 'delete', field 'user' the username making the request, 'id' set to quote id.
WARNING: This just deletes whatever quote you tell it to. Get the ID right.

returns JSON status object
a JSON object status information.
